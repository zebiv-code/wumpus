# Hunt the Wumpus

WebGL 3D implementation of Gregory Yob's classic 1973 cave game. Navigate a dodecahedral cave, avoid pits and super bats, and hunt the Wumpus with crooked arrows.

**[Live Demo](https://zebiv.com/wumpus/)**

## Features

- 3D dodecahedral cave rendered in WebGL
- Faithful recreation of the original game logic
- Visual hazard warnings (smell, draft, bats)
- Arrow path visualization
- Keyboard and click controls

## Tech

- Vanilla HTML/CSS/JavaScript
- WebGL for 3D rendering
- ES modules
- No build step, no dependencies
